## 🚀 Como rodar

### Front-end
Abra `templates/index.html` diretamente no navegador.

### Back-end (Python)
1. Instale as dependências:
```bash
pip install -r requirements.txt
```

2. Execute o servidor:
```bash
python app.py
```
