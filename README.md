### 🚀 WeekTech - 2ª Edição 2026

Bem-vindo ao repositório oficial da página de inscrição da WeekTech, um evento focado em Inteligência Artificial, dados e transformação digital. Este projeto foi desenvolvido para oferecer uma experiência imersiva aos participantes, apresentando uma interface moderna, responsiva e focada em conversão.
📸 Preview do Projeto
🛠 Tecnologias Utilizadas

Este projeto foi construído utilizando as bases fundamentais do desenvolvimento web, focando em desempenho e semântica:

    HTML5: Estruturação semântica do conteúdo, garantindo acessibilidade e organização lógica das seções (cabeçalho, navegação, herói e chamadas para ação).

    CSS3: Estilização avançada, incluindo:

        Flexbox: Para alinhamento preciso de elementos e disposição dos itens de navegação.

        Transições: Efeitos suaves para interações de botões e estados de hover.

        Tipografia e Design: Uso de estilos personalizados para criar uma hierarquia visual clara, com destaque especial para elementos chave da marca.

🎯 Objetivo do Projeto

O objetivo principal desta interface é comunicar de forma clara a proposta de valor do evento: "O futuro é a IA". A página foi desenhada para guiar o usuário através de um fluxo intuitivo, facilitando a inscrição e a consulta da programação, mantendo uma estética limpa e profissional.
